Operational UI Cookbook

Use this page to find the right helper flow by operational task rather than by raw component name. The cookbook is a workflow router: it points to the dedicated demo page and the correct shared helper instead of trying to duplicate the component workbench pages.

Common Modal Workflows

Use the narrowest helper that matches the flow. If a workflow no longer fits cleanly, drop to the lower-level modal helper rather than overloading a preset.

Login

Use createLoginFormModal(...) for shared login structure, busy submit handling, and field-name remapping.
